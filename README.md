# Tegraf Final Project - Analisis Narasi

Proyek ini adalah implementasi *Graph Theory* untuk analisis narasi di media sosial (khususnya platform X) terkait dengan kandidat Capres 2024.

## Cara Melihat Visualisasi Akhir

Visualisasi jaringan (*network graph*) dari analisis narasi telah di-generate dalam bentuk file HTML interaktif. Anda dapat membukanya dengan cara:

1. Buka folder/file explorer.
2. Navigasi ke `data/DE-sample-X-capres2024/DE-sample-X-capres2024/`.
3. Buka file **`visualisasi_finale.html`** menggunakan browser pilihan Anda (Google Chrome, Firefox, dll).

## Struktur Repositori

- **`docs/`**: Berisi seluruh file dokumentasi (`.md`), ringkasan statistik *cluster*, laporan proyek, dan presentasi terkait (`.pptx`).
- **`assets/`**: Berisi *screenshot* hasil pengujian visualisasi.
- **`data/`**: Berisi dataset *raw*, hasil pembersihan data, serta file HTML interaktif untuk visualisasi.
- **`graph_theory/`**: Modul utama untuk membangun struktur *graph*.
- **`generate_final_dashboard.py`**: Script Python utama yang digunakan untuk menghasilkan dashboard visualisasi *graph*.
- **`FP_Tegraf.py`**: Skrip utama lainnya (dapat berisi implementasi terkait).
